# Magic Cube 3D

Projeto desenvolvido utilizando HTML, CSS, JavaScript e Three.js para criar uma simulação interativa de um Cubo Mágico (Rubik's Cube) em ambiente 3D.

## Funcionalidades

- Cubo 3×3×3 composto por 27 cubinhos
- Rotação animada das faces
- Câmera 3D com controle por mouse usando OrbitControls
- Cores padrão do Cubo Mágico
- Rotação de múltiplas faces por teclado
- Contador de movimentos
- Sistema de embaralhamento
- Temas de cores customizáveis

## Controles

| Tecla | Ação |
|--------|-------|
| U | Girar face superior |
| D | Girar face inferior |
| F | Girar face frontal |
| B | Girar face traseira |
| R | Girar face direita |
| L | Girar face esquerda |

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Three.js
